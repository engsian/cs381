echo vs print

// No return value
// Slightly faster
// Can output multiple values

echo("Using parentheses");

Recommendation: Use echo — it's more common and slightly faster. Use print only when you need the return value.

trim("  hello  ");         // "hello" (remove whitespace)

Array Functions

$numbers = [3, 1, 4, 1, 5, 9];

array_filter($numbers, fn($n) => $n > 3);  // Filter
array_map(fn($n) => $n * 2, $numbers);     // Transform each

Objects

Grouping data and behavior together

What are Objects?

Array

Collection of values accessed by index or key

$user = [
    "name" => "Ahmed",
    "age" => 25
];

Object

Data + functions (methods) bundled together

$user = new User();
$user->name = "Ahmed";
$user->greet();

Key difference: Objects can have methods (functions) that operate on their data. Arrays only store data.

Creating a Simple Class

<?php
// Define a class (blueprint for objects)
class User {
    // Properties (data)
    public $name;
    public $email;

    // Constructor (runs when object is created)
    public function __construct($name, $email) {
        $this->name = $name;
        $this->email = $email;
    }

    // Method (function inside class)
    public function greet() {
        return "Hello, I'm " . $this->name;
    }
}

// Create an object (instance of the class)
$user1 = new User("Ahmed", "ahmed@example.com");
$user2 = new User("Sara", "sara@example.com");

echo $user1->greet();  // "Hello, I'm Ahmed"
echo $user2->name;     // "Sara"
?>

Why Use Objects?

Encapsulation

Bundle related data and functions together

Reusability

Create multiple instances from one class

Organization

Structure code in a logical, maintainable way

Data Protection

Control access with public/private/protected

Note: Full Object-Oriented Programming (OOP) — inheritance, interfaces, traits — is covered in a dedicated course. This is just an introduction!

GET vs POST

GET Method

page.php?name=Ahmed&age=25
  • Data visible in URL
  • Can be bookmarked
  • Limited to ~2000 characters
  • Use for: search, filters

Never for sensitive data!

POST Method

Data sent in request body
  • Data hidden from URL
  • Cannot be bookmarked
  • No size limitation
  • Use for: login, forms

Preferred for most forms

Processing Form Data

<?php
// process.php - Receives form data

// Check if form was submitted via POST
if ($_SERVER["REQUEST_METHOD"] == "POST") {

    // Access form fields using $_POST['name']
    $name = $_POST["name"];
    $email = $_POST["email"];

    echo "Hello, $name!";
    echo "Your email is: $email";
}

// For GET requests, use $_GET
// Example URL: search.php?query=PHP
if (isset($_GET["query"])) {
    $searchTerm = $_GET["query"];
    echo "You searched for: $searchTerm";
}
?>

Always check: Use isset() to verify a variable exists before using it, or use null coalescing: $name = $_POST["name"] ?? "";

Form Security

NEVER trust user input!

XSS Attack

User enters: <script>alert('hacked')</script>

If displayed without sanitizing, malicious script runs!

Protection

Use htmlspecialchars() to convert special characters to HTML entities.

< becomes &lt;

// DANGEROUS - Never do this!
echo $_POST["name"];  // Could contain malicious code

// SAFE - Always sanitize output
echo htmlspecialchars($_POST["name"], ENT_QUOTES, 'UTF-8');

The test_input() Function

A reusable function to sanitize all user input

<?php
function test_input($data) {
    $data = trim($data);           // Remove whitespace
    $data = stripslashes($data);   // Remove backslashes
    $data = htmlspecialchars($data, ENT_QUOTES, 'UTF-8');  // Convert special chars
    return $data;
}

// Usage
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $name = test_input($_POST["name"]);
    $email = test_input($_POST["email"]);
    $comment = test_input($_POST["comment"]);

    // Now safe to use these variables
    echo "Name: $name";
}
?>

Form Validation

<?php
$errors = [];
$name = $email = "";

if ($_SERVER["REQUEST_METHOD"] == "POST") {
    // Validate Name (required, letters only)
    if (empty($_POST["name"])) {
        $errors[] = "Name is required";
    } else {
        $name = test_input($_POST["name"]);
        if (!preg_match("/^[a-zA-Z-' ]*$/", $name)) {
            $errors[] = "Name can only contain letters and spaces";
        }
    }

    // Validate Email (required, valid format)
    if (empty($_POST["email"])) {
        $errors[] = "Email is required";
    } else {
        $email = test_input($_POST["email"]);
        if (!filter_var($email, FILTER_VALIDATE_EMAIL)) {
            $errors[] = "Invalid email format";
        }
    }

    // If no errors, process the form
    if (empty($errors)) {
        echo "Form submitted successfully!";
    }
}
?>

Complete Form Example

<?php
$name = $email = $message = "";
$errors = [];
$success = false;

function test_input($data) {
    return htmlspecialchars(stripslashes(trim($data)), ENT_QUOTES, 'UTF-8');

if ($_SERVER["REQUEST_METHOD"] == "POST") {
    // Validate name
    if (empty($_POST["name"])) {

    // Validate email
    if (empty($_POST["email"])) {
        $errors["email"] = "Email is required";

        // Save to database, send email, etc.

Sessions

Maintaining state across pages

What are Sessions?

The Problem

HTTP is stateless — each request is independent. Server doesn't remember previous requests.

The Solution

Sessions store user data on the server, accessible across multiple pages.


User visits

Server creates
session ID

ID stored in
browser cookie

Data stored
on server

Using Sessions

<?php
// MUST be called before any HTML output!
session_start();

// Set session variables
$_SESSION["username"] = "Ahmed";
$_SESSION["user_id"] = 123;
$_SESSION["logged_in"] = true;

// Access session variables (on any page)
echo "Welcome, " . $_SESSION["username"];

// Check if session variable exists
if (isset($_SESSION["logged_in"]) && $_SESSION["logged_in"] === true) {
    echo "You are logged in!";
} else {
    echo "Please log in.";
}

// Remove a specific session variable
unset($_SESSION["user_id"]);

// Destroy entire session (logout)
session_unset();      // Clear all session variables
session_destroy();    // Destroy the session
?>

Session Example: Login System

<?php // login.php
session_start();

if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $username = $_POST["username"];
    $password = $_POST["password"];

    // In real app: check against database with password_verify()
    if ($username === "admin" && $password === "secret123") {
        $_SESSION["logged_in"] = true;
        $_SESSION["username"] = $username;
        header("Location: dashboard.php");  // Redirect
        exit();
    } else {
        $error = "Invalid credentials";
    }
}
?>

<?php // dashboard.php (protected page)
session_start();

// Check if user is logged in
if (!isset($_SESSION["logged_in"]) || $_SESSION["logged_in"] !== true) {
    header("Location: login.php");
    exit();
}

echo "Welcome to your dashboard, " . $_SESSION["username"];
?>

<?php // logout.php
session_start();
session_unset();
session_destroy();
header("Location: login.php");
exit();
?>

Using Cookies

<?php
// setcookie() MUST be called before any HTML output!

// Set a cookie (expires in 30 days)
setcookie("username", "Ahmed", time() + (86400 * 30), "/");
//         name       value    expiration (seconds)    path

// Set a secure cookie (HTTPS only, HTTP only)
setcookie("session_token", "abc123", [
    "expires" => time() + 3600,
    "path" => "/",
    "secure" => true,      // HTTPS only
    "httponly" => true,    // Not accessible via JavaScript
    "samesite" => "Strict" // Prevent CSRF
]);

// Read a cookie
if (isset($_COOKIE["username"])) {
    echo "Welcome back, " . $_COOKIE["username"];
}

// Delete a cookie (set expiration in the past)
setcookie("username", "", time() - 3600, "/");
?>

File Handling

Reading and writing files

Reading Files

<?php
// Read entire file into a string
$content = file_get_contents("data.txt");
echo $content;

// Read file into an array (one line per element)
$lines = file("data.txt", FILE_IGNORE_NEW_LINES);
foreach ($lines as $line) {
    echo $line . "<br>";
}

// Read file line by line (memory efficient for large files)
$handle = fopen("data.txt", "r");
if ($handle) {
    while (($line = fgets($handle)) !== false) {
        echo $line;
    }
    fclose($handle);
}

// Check if file exists
if (file_exists("data.txt")) {
    echo "File exists!";
}
?>

Writing Files

<?php
// Write to file (creates or overwrites)
file_put_contents("output.txt", "Hello, World!");

// Append to file
file_put_contents("log.txt", "New entry\n", FILE_APPEND);

// Using fopen/fwrite for more control
$handle = fopen("data.txt", "w");  // w = write (overwrite)
fwrite($handle, "Line 1\n");
fwrite($handle, "Line 2\n");
fclose($handle);

// File modes:
// "r"  - Read only
// "w"  - Write only (creates/overwrites)
// "a"  - Append (creates if not exists)
// "r+" - Read and write
// "w+" - Read and write (overwrites)
?>

File Upload

<!-- HTML form for file upload -->
<form action="upload.php" method="POST" enctype="multipart/form-data">
    <input type="file" name="uploaded_file">
    <button type="submit">Upload</button>

<?php // upload.php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $file = $_FILES["uploaded_file"];

    // File information
    $fileName = $file["name"];       // Original filename
    $fileTmp = $file["tmp_name"];    // Temporary path
    $fileSize = $file["size"];       // Size in bytes
    $fileError = $file["error"];     // Error code

    // Validate and move file
    $allowedTypes = ["image/jpeg", "image/png", "application/pdf"];
    $maxSize = 5 * 1024 * 1024;  // 5MB

    if ($fileError === 0 && in_array($file["type"], $allowedTypes) && $fileSize <= $maxSize) {
        $destination = "uploads/" . basename($fileName);
        if (move_uploaded_file($fileTmp, $destination)) {
            echo "File uploaded successfully!";
        }
    } else {
        echo "Upload failed. Check file type and size.";
    }
}
?>
